# Brandon Aris Chen — Engineering Portfolio

Personal portfolio for Brandon Aris Chen, a Bachelor of Engineering Technology student in Electronics & Controls at Cape Breton University.

The site documents professional experience and selected engineering work through system context, implementation details, evidence, testing, and design revisions.

## Local development

Requires Node.js 22 or newer.

```bash
npm install
npm run dev
```

Run the production build and tests with:

```bash
npm test
npm run lint
```

## Deployment

The site is statically exported and deployed to GitHub Pages by the workflow in `.github/workflows/pages.yml`.
